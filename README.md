## Як запустити проєкт локально (без Docker)

Цей спосіб підійде для розробки та написання коду.

### 1. Клонування репозиторію
Відкрийте термінал та виконайте команду:
```bash
git clone https://github.com/AndreyShyshkin/Python-Hotel
cd Python-Hotel
```

### 2. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 3. Налаштування бази даних

Застосуйте міграції для створення необхідних таблиць у базі даних:

```bash
python manage.py migrate
```

### 4. Створення суперкористувача (для доступу в Admin-panel)

Щоб керувати номерами, категоріями та іншими даними, створіть адміністратора:

```bash
python manage.py createsuperuser
```
(Дотримуйтесь інструкцій у терміналі: введіть логін, email та пароль)

### 5. Запуск сервера розробки

```bash
python manage.py runserver
```

Після цього сайт буде доступний за адресою: http://127.0.0.1:8000/ </br>
Адмін-панель доступна за адресою: http://127.0.0.1:8000/admin/
