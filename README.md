# Веб-додаток: Сайт Готелю (CI/CD Pipeline)

Цей проєкт розроблено в рамках лабораторної роботи з дисципліни "Неперервна інтеграція та розгортання ПЗ". 
Тема: Розробка веб-застосунків із використанням принципів і інструментів Continuous Integration.
Рівень складності: Складний (Сайт для готелю зі складною системою категорій та фільтрів продуктів).

* Іващук Артем

## Технологічний стек
* **Backend:** Python, Django
* **Frontend:** HTML, CSS, JS (готовий шаблон)
* **CI/CD:** GitHub Actions, Docker, Docker Compose
* **База даних:** SQLite (за замовчуванням для локальної розробки)
